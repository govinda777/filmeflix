## Tecnologias utilizadas

* Java
* Spring
* JWT
* MongoDB
* Swagger
* Docker
* Maven

## Pré requisitos

* Docker
* Maven

## Como executar

- Passo 1 - execute o comando :

```cmd
mvn clean install
```

- Passo 2 - inicie a aplicação utilizando o docker-compose

```cmd
docker-compose up
```
